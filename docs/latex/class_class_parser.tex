\doxysubsection{Detailed Description}
Responsável pela leitura e análise estática (parsing) de um arquivo .class. 

Esta classe encapsula a lógica de ler o arquivo binário byte a byte e preencher as estruturas de dados definidas em \doxylink{classfile__types_8h_source}{classfile\+\_\+types.\+h}. Ela lida com a ordem dos bytes (Big-\/\+Endian) e a estrutura sequencial do formato .class.
